## 1. What is Closure?
A closure is when a function remembers variables
from its outer scope even after the outer function has finished executing.

## 2. What is Hoisting?
Hoisting is JavaScript's default behavior of moving
variable and function declarations to the top of their scope.

## 3. What is the Event Loop?
The event loop handles asynchronous operations
and ensures non-blocking execution in JavaScript.

## 4. Difference between == and ===
- == compares values only
- === compares value and type

## 5. What is this keyword?
`this` refers to the object that is currently calling the function.
