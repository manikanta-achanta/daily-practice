## 1. Difference between var, let, and const
- var: function scoped
- let: block scoped
- const: block scoped and cannot be reassigned

## 2. What is a Promise?
A Promise represents a value that may be available now, later, or never.

## 3. What is async/await?
Modern way to handle asynchronous code built on top of Promises.

## 4. What is the difference between map and forEach?
- map returns a new array
- forEach does not return anything

## 5. What is the DOM?
DOM stands for Document Object Model. It allows JavaScript to interact with HTML elements.
